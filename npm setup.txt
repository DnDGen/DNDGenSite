run the following commands, relevant to the version of Visual Studio you are running on your machine (i.e., 2010, 2012, 2013, etc.):

    npm install --msvs_version=<VSversion>

If still problems, you might need some global installs 

    npm install -g --msvs_version=<VSversion>

If things still aren't working, try doing some updates

    npm update
    npm update -g

If you see issues with various package installations, trying installing each one

    npm install <package> --msvs_version=<VSversion>
    npm install <package> -g --msvs_version=<VSversion>
